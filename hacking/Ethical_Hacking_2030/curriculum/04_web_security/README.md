# Module 4: Web Security / Module 4: Bảo Mật Web

## Overview / Tổng Quan
This module covers web application security, including common vulnerabilities, attack techniques, and defense strategies. Students will learn how to identify, exploit, and prevent web security issues.

Module này bao gồm bảo mật ứng dụng web, bao gồm các lỗ hổng phổ biến, kỹ thuật tấn công và chiến lược phòng thủ. Học viên sẽ học cách xác định, khai thác và ngăn chặn các vấn đề bảo mật web.
