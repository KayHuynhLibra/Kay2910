# Module 5: Network Security / Module 5: Bảo Mật Mạng

## Overview / Tổng Quan
This module covers network security principles, protocols, and practices. Students will learn about network attacks, defense mechanisms, and security tools used to protect network infrastructure.

Module này bao gồm các nguyên tắc, giao thức và thực hành bảo mật mạng. Học viên sẽ học về các cuộc tấn công mạng, cơ chế phòng thủ và công cụ bảo mật được sử dụng để bảo vệ hạ tầng mạng.
